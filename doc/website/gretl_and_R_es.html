gretl (página principal aquí) está diseñado como un paquete econométrico muy sencillo para el usuario. Aunque también es razonablemente sofisticado, le faltan algunos de los métodos estadísticos especializados que un analísta econométrico puede desear.

Como una alternativa ante esta limitación, gretl ofrece un interfaz que da acceso al paquete estadístico de software libre GNU R. De acuerdo al resumen que hay en el sitio web del proyecto R, "R es `GNU S' - Un lenguaje y un entorno para el cálculo estadístico y gráficos. R es similar al tantas veces premiado sistema S, que fue desarrollado en los laboratorios Bell por John Chambers y otros."

Gretl ha tenido desde hace tiempo la posibilidad de guardar el conjunto de datos actual en un formato adecuado para el análisis con R; o también la opción ("Iniciar GNU R" bajo el menú 'sesión' de gretl) de lanzar una sesión de R con el conjunto de datos actual de gretl cargado automáticamente en el espacio de trabajo de R. Sin embargo, desde la versión 1.7.5 de Gretl, su interacción con R es mucho más soficticada: ahora es posible incorporar guiones de R dentro de los guiones de Gretl y que ambos programas se transfieran los datos y los resultados de una manera relativamente trasnparente. Para más detalles, la guía Gretl User's Guide, que se suministra junto con el programa y que también puede descargarse, tiene un capítulo dedicado a este tema.

Para encontrar más información sobre GNU R le recomiendo empezar con An Introduction to R, disponible aquí como fichero PDF (Hay una version en español en R-intro-1.1.0-espanol.1.pdf). Para más información y para descargar R, visite cran.r-project.org.
