Gnu Regression, Econometrics and Time-series Library

Versione MS Windows

Download

gretl per Windows è disponibile sotto forma di file eseguibile auto-estraente. gretl_install.exe (2 febbraio 2004, circa 7MB). Basta scaricarlo ed eseguirlo: apparirà una finestra di dialogo per scegliere dove installare il programma (la cartella predefinita è c:\userdata\gretl). Il programma è stato testato su Windows 98 e Windows XP, ma dovrebbe funzionare bene su tutte le versioni di Windows superiori a Windows 95.

Programmi opzionali che è possibile installare:

X-12-ARIMA (correzione stagionale, modelli ARIMA)

x12a_install.exe

TRAMO/SEATS (correzione stagionale, modelli ARIMA)

ts_install.exe

Dati da Wooldridge (dataset aggiuntivi dal libro omonimo)

wooldridge_data.exe

Dati da Gujarati (dataset aggiuntivi dal libro omonimo)

gujarati_data.exe

Dati da Econometric Theory and Methods di Davidson e MacKinnon

ETM_data.exe

I file precedenti sono tutti auto-estraenti e vanno installati dopo aver installato gretl.

Il manuale di gretl è disponibile separatamente sotto forma di file PDF, in versione italiana (manual_it.pdf) o inglese (manual-a4.pdf, con pagine in formato A4, o manual.pdf, con pagine in formato US letter). Un programma gratuito per visualizzare i file PDF può essere scaricato dal sito Adobe.

I file eseguibili sono stati cross-compilati su GNU/Linux usando mingw32 e gtk for win32 di Tor Lillqvist. Per il programma di installazione si ringrazia Jordan Russell.

Aggiornare gretl

Quando si aggiorna un'installazione esistente di gretl, è una buona idea eseguire per prima cosa il programma di disinstallazione unins000.exe, che si trova nella cartella principale di gretl. In questo modo si eviteranno possibili problemi dovuti a "vecchie" DLL o impostazioni del registro di Windows. Si noti che eseguire la disinstallazione non rimuoverà nessuno dei file creati nella cartella gretl/user, ma cancellerà le personalizzazioni impostate attraverso il menù Preferenze, che quindi andranno reimpostate una volta avviata la nuova versione del programma.

Eseguire gretl via rete

È disponibile una pagina di istruzioni per configurare gretl per windows in modo da essere eseguito via rete, ad esempio in un laboratorio informatico.

Compilare gretl per Windows

Se si intende compilare da sé gretl per Windows, basta scaricare l'ultima versione del pacchetto sorgente di gretl e guardare nella directory win32, dove si trova un Makefile per la versione Windows. Il file è progettato per cross-compilare gretl su Linux, ma non dovrebbe essere difficile modificarlo per compilare il programma su Windows, in particolare se si usa il compilatore mingw32.
