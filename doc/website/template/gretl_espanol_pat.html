Es un paquete de software para análisis econométrico escrito en el lenguaje de programación C. Es software libre y de código abierto. Vd. puede redistribuirlo y/o modificarlo bajo los términos de la Licencia Pública General GNU(GPL) según se ha especificado por la Free Software Foundation.

Características

  • Interface fácil e intuitiva (ahora en Español, Francés, Italiano, Polaco y Alemán además del Inglés)

  • Incluye una gran variedad de estimadores basados en mínimos cuadrados y máxima verosimilitud, entre ellos, estimadores de sistemas de ecuaciones

  • Tiene instrucciones sencillas para realizar cosas como el contraste de Dickey-Fuller aumentado, el contraste de estabilidad estructural de Chow, autorregresiones vectoriales o estimación de modelos ARMA.

  • Los resultados de los modelos se pueden guardar como ficheros LaTeX, en formato tabular y/o de ecuación.

  • Incluye un lenguaje de programación vía 'scripts' (guiones de instrucciones): las órdenes se pueden introducir por medio de los menús o por medio de guiones.

  • Estructura de bucles de instrucciones para simulaciones de Monte Carlo y procedimientos de estimación iterativos.

  • Controlador gráfico mediante menús, para el ajuste fino de los gráficos Gnuplot

  • Enlace a GNU R para análisis más sofisticados de los datos

Formatos de datos

Lee sus propios ficheros de datos en formato XML, ficheros CSV (Comma separated values), los de las hojas de cálculo Excel y Gnumeric, ficheros .dta de Stata, ficheros de trabajo de Eviews, su propio formato binario de bases de datos (que permite frecuencias mixtas de los datos y larguras mixtas de las series) y bases de datos RATS4. Incluye una base de datos macroeconómicos de los EEUU. Ver también la página datos para gretl.

Manual

El manual de Gretl viene en dos ficheros PDF, la Guía del Usuario y la Guía de Instrucciones. La versión inglesa de ambas, formateada para el tipo de papel letter de EE.UU., está incluída en el paquete fuente de Gretl y en las distribuciones binarias. Gretl accederá a las otras versiones disponibles --es decir, la versión en ingles en papel a4, y las versiones en Italiano y en Español-- bajo demanda, via Internet. Los ficheros de manual también están disponibles aquí http://ricardo.ecn.wfu.edu/pub//gretl/manual/ (nota del traductor: la versión en español suele llevar bastante retraso. Para entender el funcionamiento de las características más novedosas del programa es recomendable utilizar la versión en Inglés).

Si Vd. no tiene el software para leer ficheros PDF puede descargarlo de forma gratuita desde el sitio Web de Adobe.

Contribuir al desarrollo de Gretl

Tiene Vd. alguna idea o código con el cual le gustaría contribuir al desarrollo de Gretl? Si es así, estaríamos encantados de escucharle. La única condición respecto al código que se incorpora a Gretl es que Vd. debe estar conforme en que aparezca bajo la Licencia Pública General GNU (GPL), es decir, Vd. debe de estar de acuerdo en que se distribuya el código fuente.

Como se ha mencionado más arriba, Gretl está escrito (principalmente) en el lenguaje de programación C, el lenguaje más común en el mundo del software de código abierto. Sin embargo, no todo el mundo se siente cómodo al escribir en C. Debe entender que esto no es una barrera para contribuir a este proyecto con código o algoritmos. Si Vd. puede proporcionar un algoritmo bien especificado para llevar a cabo alguna interesante tarea econométrica, probablemente no nos será difícil reescribirlo en C o hacer que Gretl lo interprete. Esto incluye código en Fortran o C++; código escrito en algún lenguaje de alto nivel para manipulación de matrices tal como Ox, Gauss, Octave, Matlab o R; e incluso pseudo-código, es decir un algoritmo explicado explícitamente, paso a paso, pero no escrito en ningún lenguaje de programación en concreto.
