Gretl reside en

Es un paquete de software para análisis econométrico escrito en el lenguaje de programación C. Es software libre. Vd. puede redistribuirlo y/o modificarlo bajo los términos de la Licencia Pública General GNU(GPL) según se ha especificado por la Free Software Foundation.

Características

  • Interface fácil e intuitiva (ahora en Francés y Español, además del Inglés)

  • Incluye una gran variedad de estimadores basados en mínimos cuadrados (entre ellos mínimos cuadrados en dos etapas)

  • Tiene instrucciones sencillas para realizar cosas como el contraste de Dickey-Fuller, el contraste de estabilidad estructural de Chow o autorregresión vectorial.

  • Los resultados de los modelos se pueden guardar como ficheros LaTeX, en formato tabular y/o de ecuación.

  • Lenguaje de programación vía 'scripts' (guiones de instrucciones) incorporado: las órdenes se pueden introducir por medio de los menús o por medio de guiones.

  • Estructura de bucles de instrucciones para simulaciones de Monte Carlo.

  • Controlador gráfico mediante menús, para el ajuste fino de los gráficos Gnuplot

  • Enlace a GNU R para análisis más sofisticados de los datos

Formatos de datos

Lee sus propios ficheros de datos en formato XML, ficheros CSV (Comma separated values), los de las hojas de cálculo Excel y Gnumeric, BOX1, su propio formato binario de bases de datos (que permite frecuencias mixtas de los datos y larguras mixtas de las series) y bases de datos RATS4. Incluye una base de datos macroeconómicos de los EEUU y un guión perl para crear una base de datos a partir de economagic.com. Ver también la página datos para gretl.

Cosas que faltan

  • No hay mucho aún sobre métodos de máxima verosimilitud (sólo logit y probit).

  • Dígame Vd. mismo qué mas cosas faltan.

Descarga

Aquí están las últimas fuentes del paquete: gretl-VERSION.tar.bz2 (c. 3Mb). Esto incluye muchos ficheros de datos y ejemplos de guiones de instrucciones.

Aquí hay un RPM binario para Red Hat Linux 7.3 i386: gretl-VERSION-1rh73.i386.rpm y un rpm binario más genérico, que tiene menos dependencias: gretl-VERSION-1generic.i386.rpm. También hay archivos binarios RPM para los programas X-12-ARIMA x12a-0.2.10-1.i386.rpm y TRAMO/SEATS tramo-seats-98-1.i386.rpm, que se pueden utilizar junto con gretl.

Dirk Eddelbuettel mantiene los paquetes de gretl para Debian GNU/Linux, que se pueden encontrar en packages.debian.org; hay más información disponible en packages.qa.debian.org.

Tanto si Vd. va a compilar gretl desde el código fuente como si va a instalar un fichero binario pre-compilado, es posible que necesite o desee comprobar las dependencias de gretl.

Gretl llama al programa 'gnuplot' para generar los gráficos. La última versión oficial de gnuplot es la 3.7.3 (fechada en 2002), que está disponible en www.gnuplot.info. Sin embargo, si Vd. va a utilizar la versión Linux de gretl en Español o Francés encontrará que dicha versión de gnuplot borra los caracteres acentuados de los gráficos. Puede obtener una versión actualizada no oficial de gnuplot que corrige este problema en: (en formato binario rpm) gnuplot-3.8j0_ac-1.i386.rpm. Como alternativa, puede también contruir Vd. mismo una versión actualizada de gnuplot a partir del código fuente del CVS: ver www.gnuplot.info para más detalles.

Aquí se puede encontrar una versión de gretl para MS Windows.

Manual

Hay un manual disponible en formato PDF (en Inglés) (900Kb, aprox. 70 páginas), tanto en tamaño 'US letter' (manual.pdf) como en tamaño A4 (manual-a4.pdf). Si Vd. no tiene el software para leer ficheros PDF puede descargarlo de forma gratuita desde el sitio Web de Adobe.
