Gretl: Gnu Regression, Econometrics and Time-series Library

Versione MS Windows

Download

gretl è disponibile per Windows sotto forma di file eseguibile auto-installante, gretl_install.exe (SHORTDATE, circa 7MB). È sufficiente scaricarlo, eseguirlo e scegliere la posizione dove verrà installato il programma (il valore predefinito è c:\userdata\gretl). Il programma è testato su Windows 98 e Windows XP; per quanto ne sappia, dovrebbe funzionare bene su tutte le versioni di Windows successive a Windows 95.

Il link precedente punta all'ultima versione ufficiale del programma, ma è anche possibile scaricare la versione "snapshot" attuale di gretl dalla Wake Forest University: gretl_install.exe. Questa versione è di solito più recente e contiene correzioni di errori (o a volte nuovi errori!), e potrebbe non essere completamente tradotta nelle lingue diverse dall'inglese.

Tutti i file precedenti sono auto-estraenti: per installarli è necessario prima installare gretl.

Il manuale completo di gretl è disponibile come file PDF (circa 1.6Mb e 100 pagine), manual_it.pdf. È possibile scaricare il software per leggere i file PDF dal sito Adobe.

Gli eseguibili sono stati cross-compilati su GNU/Linux usando mingw32 e gtk per win32 di Tor Lillqvist. Il programma di installazione è di Jordan Russell.

Aggiornare gretl

Prima di aggiornare gretl è consigliabile eseguire il programma di disinstallazione unins000.exe, che si trova nella cartella principale di gretl. Questo evita possibili strani comportamenti dovuti a "vecchie" DLL o impostazioni del registro di Windows. Attenzione: disinstallando il programma non verranno rimossi eventuali file creati nella cartella gretl/user, ma verranno cancellate le personalizzazioni fatte nel menù "Preferenze", che quindi dovranno essere reimpostate una volta avviata la nuova versione del programma.

Eseguire gretl via rete

Sono disponibili istruzioni per configurare gretl per Windows in modo da essere eseguito via rete, ad esempio in un laboratorio informatico.

Compilare gretl per Windows

Per compilare gretl per Windows, basta scaricare il pacchetto dei sorgenti di gretl (si veda qui per la versione più recente) e dare un'occhiata alla directory win32. Il Makefile per la versione Windows è in realtà scritto per cross-compilare gretl su Linux, ma non dovrebbe essere difficile modificarlo per compilare il programma direttamente su Windows, in particolare se si usa il compilatore mingw32.
