Gretl: Gnu Regression, Econometrics and Time-series Library

Versione MS Windows

Download

gretl è disponibile per Windows sotto forma di file eseguibile auto-installante, gretl-VERSION.exe (SHORTDATE, circa 9MB). È sufficiente scaricarlo, eseguirlo e scegliere la posizione dove verrà installato il programma (il valore predefinito è c:\Program Files\gretl). Il programma è stato testato principalmente su Windows XP; dovrebbe funzionare bene su tutte le versioni di Windows basate su NT (ossia, NT4, 2000, XP o Vista, ma non Win9X o Windows ME).

Il link precedente punta all'ultima versione ufficiale del programma, ma è anche possibile scaricare la versione "snapshot" attuale di gretl dalla Wake Forest University: gretl_install.exe. Questa versione è di solito più recente e contiene correzioni di errori (o a volte nuovi errori!), e potrebbe non essere completamente tradotta nelle lingue diverse dall'inglese.

Se si vuole usare gretl su versioni di Windows meno recenti, è possibile usare la versione 1.6.5 del programma (richiede Windows 98 o superiore).

Gli eseguibili sono stati cross-compilati su GNU/Linux usando mingw32 e gtk per win32 di Tor Lillqvist. Il programma di installazione è di Jordan Russell.

Supporto multilingua

Oltre alla lingua inglese, gretl supporta il francese, l'italiano, lo spagnolo, il tedesco, il basco, il portoghese e il polacco. Di solito non occorre fare nulla di speciale per poter usare gretl nella lingua locale: il programma dovrebbe usare automaticamente le traduzioni disponibili basandosi sull'impostazione della lingua attiva sul computer. In alcune circostanze, però, potrebbe essere necessario modificare l'impostazione della lingua sul computer, usando il pannello di controllo di Windows. Se si incontrano problemi in questa operazione, ecco un controllo da fare:

  • Aprire un terminale di Windows (o quello che a volte viene chiamato "Prompt di MS/DOS").

  • Spostarsi (cd) nella directory dove gretl è stato installato. Ad esempio, scrivere

    cd c:\programmi\gretl

    e premere Invio.

  • Scrivere "set LANG=" seguito dal codice a due lettere della propria lingua. Ad esempio, per la lingua italiana:

    set LANG=it

    e premere Invio. Per il francese occorrerà usare fr, per lo spagnolo es, per il polacco pl e per il tedesco de.

  • Quindi, avviare gretl dal terminale, scrivendo

    gretlw32.exe

    e premendo Invio.

Gretl dovrebbe apparire nella lingua selezionata, se essa è supportata. Se si riesce a far funzionare gretl nella propria lingua seguendo questa procedura, dovrebbe essere possibile ottenere lo stesso risultato anche attraverso le impostazioni della lingua nel pannello di controllo di Windows.

Aggiornare gretl

Prima di aggiornare gretl è consigliabile eseguire il programma di disinstallazione unins000.exe, che si trova nella cartella principale di gretl. In questo modo si eviteranno possibili problemi dovuti a "vecchie" DLL o impostazioni del registro di Windows. Attenzione: disinstallando il programma non verranno rimossi eventuali file creati nella cartella gretl/user, ma verranno cancellate le personalizzazioni fatte nel menù "Preferenze", che quindi dovranno essere reimpostate una volta avviata la nuova versione del programma.

Eseguire gretl via rete

Sono disponibili istruzioni per configurare gretl per Windows in modo da essere eseguito via rete, ad esempio in un laboratorio informatico.

Compilare gretl per Windows

Per compilare gretl per Windows, basta scaricare il pacchetto dei sorgenti di gretl (si veda qui per la versione più recente) e dare un'occhiata alla directory win32. Il Makefile per la versione Windows è in realtà scritto per cross-compilare gretl su Linux, ma non dovrebbe essere difficile modificarlo per compilare il programma direttamente su Windows, in particolare se si usa il compilatore mingw32.
