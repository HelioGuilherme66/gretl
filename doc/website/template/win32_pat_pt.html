Gretl: Gnu Regression, Econometrics and Time-series Library

Versão MS Windows

Download

O Gretl para Windows é fornecido na forma de um ficheiro/arquivo de auto-instalação, gretl-VERSION.exe (SHORTDATE, cerca de 12MB). Basta fazer o download e executá-lo: o instalador perguntará qual o local para instalação do programa (a localização padrão é c:\Arquivo de programas\gretl). O programa tem sido testado majoritariamente no Windows XP; porém deverá funcionar corretamente em todas as versões NT do Windows (isto é, NT4, 2000, XP ou Vista; mas não no Win9X ou Windows ME).

O link acima corresponde à versão oficial mais recente. Outra possibilidade é fazer o download da imagem mais recente ("snapshot") do Gretl: gretl_install.exe. Ao optar por instalar o "snapshot", você terá, possivelmente, uma versão mais actualizada do Gretl. Esta versão conterá algumas resoluções de bugs, porém, dado seu caráter experimental, o "snapshot" poderá introduzir novos bugs. Além disso, traduções para idiomas diferentes do Inglês poderão estar incompletos. Para saber quais novidades foram introduzidas no "snapshot", consulte o Registro de alterações (no link "em progresso").

Se desejar executar o Gretl em versões mais antigas do Windows você pode usar a versão 1.6.5 do programa (requer Windows 98 ou superior).

Todos os ficheiros/arquivos acima são de auto-instalação. O Gretl deve ser instalado antes dos pacotes extras.

Os executáveis foram compilados no GNU/Linux utilizando o mingw32 e o GTK para Windows (obrigado Tor Lillqvist!). O instalador é uma cortesia de Jordan Russell.

Suporte a várias idiomas

Além do Inglês, o Gretl está disponível em Francês, Italiano, Espanhol, Alemão, Basco, Português, Polaco/Polonês, Turco e Russo. Normalmente não será necessário fazer nada em especial para que o Gretl seja exibido no idioma local: o programa vem com os conjuntos completos das traduções, que devem ser activadas automaticamente com base no idioma em uso no computador. No entanto, em certas circunstâncias, deverá ser necessário ajustar a sua língua para obter o efeito desejado. Isto pode ser feito no Painel de Controle do Windows.

Actualizando o Gretl

Antes de actualizar uma instalação do Gretl, recomenda-se primeiro desinstalá-lo executando o programa unins000.exe, que se encontra na pasta principal do Gretl. Isto assegurará que não haverá comportamentos estranhos devidos a DLLs "velhos" ou valores inadequados no registo do Windows. Note que ao executar o desinstalador o programa não irá afectar ficheiros/arquivos criados na pasta gretl/user, mas irá apagar as customizações que tenham sido feitas usando o menu "Preferências", portanto estas terão que ser redefinidas quando se iniciar a nova versão do programa.

Existe uma página de conselhos (em Inglês) neste link para quem pretender configurar o Gretl para Windows de modo a executá-lo em rede, como por exemplo num laboratório de informática.

Compilando o Gretl para Windows

Para compilar o Gretl para Windows, faça o download do pacote de código fonte do gretl (ver aqui a versão mais recente) e veja a/o directoria/diretório win32. Você encontrará o "Makefile" para a versão Windows, que na realidade foi escrito para compilação cruzada do Gretl em Linux, mas não deverá ser muito difícil alterá-lo para o uso em Windows, particularmente o compilador mingw32 estiver sendo utilizado.
