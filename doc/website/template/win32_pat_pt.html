Gretl: Gnu Regression, Econometrics and Time-series Library

Versão MS Windows

Download

gretl para Windows está na forma de um ficheiro de auto-instalação, gretl-VERSION.exe (SHORTDATE, cerca de 7MB). Basta fazer o download e executá-lo e responder ao local da instalação, (o valor predefinido é c:\userdata\gretl). O programa está testado em  Windows 98 e Windows XP; pelo aquilo que sei, deverá funcionar bem em todas as versões do Windows que sucederam ao Windows 95.

A ligação atrás apresentada é para a versão oficial mais recente. Outra possibilidade é fazer o download da versão actual "snapshot" do gretl a partir da Wake Forest University: gretl_install.exe. Esta versão deverá estar mais actualizada: por vezes terá correcções de erros, e noutras vezes poderá ter erros recentemente introduzidos! Traduções noutras línguas que não o Inglês, é de esperar que não estejam completas.

Extras opcionais que pode querer instalar:

X-12-ARIMA (ajustamento com sazonalidade, modelos ARIMA)

x12a_install.exe

TRAMO/SEATS (ajustamento com sazonalidade, modelos ARIMA)

ts_install.exe

Conjuntos de Dados para Introductory Econometrics de Wooldridge

wooldridge_data.exe

Conjuntos de Dados para Basic Econometrics de Gujarati

gujarati_data.exe

Conjuntos de Dados e script para Introduction to Econometrics de Stock e Watson.

stock_watson.exe

Conjuntos de Dados para Econometric Theory and Methods de Davidson e MacKinnon

ETM_data.exe

Todos os ficheiros acima são de auto-instalação. O gretl tem que ser instalado antes de instalar pacotes extras.

O manual completo está disponível separadamente em formato PDF: manual.pdf. Está em tamanho de papel E.U.A.; também há uma versão em tamanho A4: manual-a4.pdf. Se não tem o software para visionar ficheiros PDF, pode fazer o download a partir do sítio de Internet da Adobe.

Os executáveis foram criados em compilação cruzada (cross-compiled) em GNU/Linux usando mingw32 e gtk para win32 de Tor Lillqvist. O programa livre de instalação é de Jordan Russell.

Suporte para várias línguas

Para além do Inglês, gretl está em Francês, Italiano, Espanhol, Alemão e Polaco. Normalmente não deve ser necessário fazer nada em especial para o programa aparecer na Língua local: o programa vem com os conjuntos completos das traduções, e elas devem ser activadas automaticamente, baseadas na Língua activa no computador. No entanto, em certas circunstâncias, deverá ser necessário ajustar a sua Língua para obter o efeito desejado. A definição da Língua do utilizador pode ser feita no Painel de Controle do Windows. Se não conseguir fazer isso funcionar, eis uma lista de verificações que pode seguir:

  • Abri uma janela de comandos do Windows (por vezes erradamente chamada de  "Janela de MS/DOS").

  • Mudar para a directoria (cd) onde o gretl está instalado. Por exemplo, escrever

    cd c:\programmi\gretl

    e premir Enter.

  • Escrever "set LANG=" seguido pelo código de duas letras da sua Língua. Por exemplo, para a Língua portuguesa:

    set LANG=pt

    e premir Enter. (Francês é fr, Italiano é it, Espanhol é es, Alemão é de,  e Polaco é pl)

  • De seguida iniciar o gretl a partir da linha de comandos, escrevendo

    gretlw32.exe

    e premindo Enter.

Gretl deverá aparecer na Língua seleccionada, se suportada. Se deste modo não for possível colocar o gretl a funcionar na Língua correcta, deve ser possível corrigir esta situação indo ao Painel de Controle do Windows e escolhendo lá a Língua correcta.

Actualizar gretl

Antes de actualizar uma instalação do gretl, convirá primeiro desinstalá-la executando para isso o programa unins000.exe, que se encontra na pasta principal do programa gretl.  Isto assegurará que não haverá comportamentos estranhos devidos a DLLs "velhos" ou valores desadequados no registo do Windows. Atenção: desinstalar o programa não irá afectar nenhuns ficheiros que o utilizador tenha criado na pasta gretl/user, mas irá apagar as customizações que tenham sido feitas usando o menu "Preferences", portanto estas terão que ser redefinidas quando se iniciar a nova versão do programa.

Executar gretl numa rede

Existe uma página de conselhos nesta ligação para quem pretender configurar o gretl para Windows de modo a executar em rede, como por exemplo num laboratório de informática.

Compilar gretl para Windows

Para para compilar o gretl para Windows, fazer o download do pacote de código fonte do gretl (ver aqui a versão mais recente) e dar uma olhadela na directoria  win32. O Makefile para a versão Windows foi escrito na realidade para compilação cruzada do gretl em Linux mas não deverá ser muito difícil de alterar para o uso em Windows, em particular se se estiver a usar o compilador mingw32.
