Gretl: Gnu Regression, Econometrics and Time-series Library

for Mac OS X

First you need to determine if your Mac has an Intel or a PPC processor. Newer Macs (since 2006) all run Intel. If you have an older Mac and are not sure, click on the Apple icon at the top left of your screen and select "About this Mac". In the window that appears you'll find the OS X version number and also the processor type.

If you have an intel Mac running OS X 10.6 (Snow Leopard) or higher, we recommend the "quartz" download below. Some users may prefer to use the X11 version of gretl.

When you're ready, select one of the links below.

  • Intel Mac, quartz (recommended)
  • Intel Mac, X11
  • PPC Mac (old)

Important

Some Mac users reported that trying to install gretl on recent OSX versions could trigger a security alert, and show a message similar to this one:

If you downloaded your dmg file from this website, you have tothing to worry about. Here's a to-do list to circumvent the problem:
  1. Open System Preferences and navigate to "Security & privacy". Select tab General.
  2. Double click .pkg file. You get a message "gretl-quartz.pkg can't be opened because Apple cannot check it ...". Click OK. to close the window.
  3. . In the tab mentioned in item 1 you will see a message with a button Open Anyway. Click that button.
  4. Double click the gretl pkg file again. Now you will get the same window as in item 1. The window has two buttons: Open and Cancel. Click Open and Gretl will be installed.
