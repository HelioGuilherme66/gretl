Gretl: Gnu Regression, Econometrics and Time-series Library

para Mac OS X sobre Intel (versión X11)

Paso 0: instale X11, si es necesario (sólo la primera vez)

Si está ejecutando OS X 10.8 (Mountain Lion), y no tiene instalado X11, tendrá que hacer esto primero. (X11 venía instalado por defecto en OS X 10.6 y 10.7.) Puede obtener X11 desde macosforge.

Paso 1: instale GTK (sólo la primera vez)

Se necesitan algunas bibliotecas adicionales para mantener gretl, en concreto GTK y algunas relacionadas. Si está ejecutando OS X 10.6 (Snow Leopard) o superior, instale GTK usando GTK2-framework.pkg. Puede ver algunas notas sobre este paquete en GTK.pdf.

Si Vd está ejecutando una versión más vieja de OS X (10.5 o inferior) debería instalar esta versión de GTK (o esta descarga alternativa). Cuidado: Esta versión de GTK no funcionará con OS X 10.7 (Lion).

Paso 2: instale gretl

Ahora para instalar el propio gretl: descargue e instale la última versión publicada o la versión "instantánea". La versión "instantánea" suele estar más actualizada: normalmente contendrá correcciones de errores pero a veces tendrá también introducidas nuevas prestaciones y con ellas, nuevos errores. Para ver qué hay de nuevo en la versión "instantánea" eche un vistazo al Historial de cambios de gretl (la versión "in progress").

ültima versión publicada (SHORTDATE)

gretl-VERSION-x11.dmg

O versión "instantánea" actual

gretl-x11.dmg

Después de la descarga, pulse dos veces sobre el fichero dmg para montar la imagen de disco. Abra la imagen de disco en el buscador (Finder) y arrastre Gretl a Aplicaciones. Pero tenga en cuenta: si está actualizando una instalación de gretl preexistente, primero debería abrir Aplicaciones en el buscador (Finder) y borrar el elemento Gretl antiguo.

Una vez que ha acabado de instalar gretl puede extraer la imagen de disco y borrar el fichero dmg.

¿Problemas?

Hay documentación sobre la imagen de disco de gretl en README.pdf, que contiene algunas sugerencias para evitar problemas. En breve, si Vd. quiere averiguar lo que ocurre con un gretl que no funciona sobre OS X, por favor intente esto: abra una ventana de terminal X11; cambie el directorio (cd) a Gretl.app (donde instaló el paquete); cambie el directorio a Contents/Resources/bin y ejecute la instrucción

./gretl

esto debería de darle algo de infromación sobre lo que no funciona.

Extras opcionales

Puede que le interese instalar X-12-ARIMA y/o TRAMO/SEATS. Estos son programas especializados en el análisis de series temporales estacionales, y aquí tiene una versión compatible con gretl.
