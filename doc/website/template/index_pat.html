Gretl is hosted by

Is a cross-platform software package for econometric analysis, written in the C programming language. It is is free software. You may redistribute it and/or modify it under the terms of the GNU General Public License (GPL) as published by the Free Software Foundation.

Features

  • Easy intuitive interface (now in French, Italian and Spanish as well as English)

  • A wide variety of least-squares based estimators, including two-stage least squares and nonlinear least squares

  • Single commands to launch things like augmented Dickey-Fuller test, Chow test for structural stability, Vector Autoregressions, ARMA estimation

  • Output models as LaTeX files, in tabular or equation format

  • Integrated scripting language: enter commands either via the gui or via script

  • Command loop structure for Monte Carlo simulations and iterative estimation procedures

  • GUI controller for fine-tuning Gnuplot graphs

  • Link to GNU R for further data analysis

Data formats

Reads own format XML data files, Comma Separated Values files, Excel and Gnumeric worksheets, BOX1 files, own format binary databases (allowing mixed data frequencies and series lengths) and RATS 4 databases. Includes a sample US macro database. See also the gretl data page.

Whether you're compiling from source or installing a pre-compiled binary, you may want/need to check out gretl's dependencies.

Gretl calls gnuplot to generate graphs. The latest official release of gnuplot is version 4.0 (dated April 2004), which is available via www.gnuplot.info.

A version of gretl for MS Windows can be found here.

Gretl for Mac OS X can be found here.

Manual

A comprehensive manual is available in PDF format (c. 1Mb, around 90 pages), in both US letter size (manual.pdf) and A4 size (manual-a4.pdf). An Italian translation of the command reference section of the manual is also available in PDF (cmdbook_it.pdf), thanks to Cristian Rigamonti. If you don't have the software to read PDF files you can download it for free from the Adobe website.
