Gretl: Gnu Regression, Econometrics and Time-series Library

per macOS

Questa pagina descrive gretl per macOS 10.6 ("Snow Leopard", uscita nel 2009) o seguente. Se la vostra versione di OS X è precedente, date un'occhiata qui.

Dovete decidere se volete la versione ufficiale o la "snapshot". La snapshot è la versione su cui il team di sviluppo sta lavorando, e quindi è più aggiornata: spesso contiene la correzione di alcuni bug, ma a volte potrebbe contenerne di nuovi. C'è una descrizione del contenuto della snapshot al link Change log (sotto la voce "in progress").

Consiglio: Se state aggiornando gretl, potrebbe essere il caso di disinstallare la vecchia versione prima di installare la nuova.

Importante: problemi di sicurezza (o presunti tali)

Alcuni utenti Mac ci hanno fatto sapere che con versioni recenti di OSX, l'istallazione di gretl non viene completata, a causa di presunti problemi di sicurezza. Se avete scaricato gretl da questo sito, non c'è da preoccuparsi. Ecco come procedere:

  1. Aprite System Preferences e cercate "Security & privacy". Selezionate il tab "General".
  2. fate doppio clic sul file .pkg. Apparirà un messaggio del tipo "gretl-quartz.pkg can't be opened because Apple cannot check it ...". Cliccate OK per chiudere la finestra.
  3. . Nel tab descritto sopra Vedrete un messaggio con un bottone "Apri comunque". Cliccatelo.
  4. Fate di nuovo doppio clic sul file pkg. Dovrebbe apparire la stessa finestra di prima, con due bottoni: Apri e Cancella. Cliccate Apri dovrebbe andare tutto a posto.

Extra opzionali

Se lavorate con serie storiche, potrebbero farvi comodo X-13-ARIMA-SEATS e/o TRAMO/SEATS, che sono programmi specializzati per l'analisi di serie stagionali. Per installare delle versioni a 64 bit compatibili con macOS, basta aprere la console di gretl e digitare i seguenti comandi:

pkg install x13as-mac64.tar.gz
pkg install tramo-seats-mac64.tar.gz
