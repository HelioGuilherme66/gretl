Gretl: Gnu Regression, Econometrics and Time-series Library

Versión para MS Windows

Descarga

Gretl para Windows viene en la forma de un fichero ejecutable auto-instalable, gretl-VERSION.exe (SHORTDATE, aproximadamente 9MB). Vd. sólo tiene que descargarlo y ejecutarlo. Durante el proceso de instalación se le preguntará por el directorio donde desea instalar el programa (por defecto es c:\Archivos de programa\gretl). El programa ha sido cheqeuado sobre Windows 98 y Windows XP; y debería funcionar bien sobre cualquier versión de Windows posterior a Windows 95.

El enlace de arriba señala a la última versión oficial del programa. Otra posibilidad es descargar la versión "instantánea" de gretl desde el servidor de la Wake Forest University: gretl_install.exe. Ésta estará normalmente más actualizada: incorporará nuevas características a prueba, a veces incorporará correcciones de errores, y otras veces ¡puede incorporar nuevos errores que antes no existían! Además es probable que las traducciones a idiomas diferentes del Inglés estén incompletas en esta versión.

Opciones extra que es posible que Vd. desee instalar:

X-12-ARIMA (corrección de estacionalidad, modelos ARIMA)

x12a_install.exe

TRAMO/SEATS (corrección de estacionalidad, detección de outliers, interpolación de valores ausentes, modelos ARIMA)

ts_install.exe

Conjuntos de datos del libro de Wooldridge, Introductory Econometrics

wooldridge_data.exe

Conjuntos de datos del libro de Gujarati, Basic Econometrics

gujarati_data.exe

Conjuntos de datos + guiones del libro de Stock and Watson, Introduction to Econometrics

stock_watson.exe

Conjuntos de datos del libro de Davidson y MacKinnon, Econometric Theory and Methods

ETM_data.exe

Todos los de arriba son ficheros auto-instalables. Vd. debe instalar Gretl antes de instalar las opciones extra.

Los ficheros ejecutables se compilaron bajo GNU/Linux usando mingw32 y el gtk para win32 de Tor Lillqvist. El programa de instalación, también software libre, es cortesía de Jordan Russell.

Soporte para varios idiomas

Ademas del Inglés, Gretl soporta el Francés, Italiano, Castellano, Alemán, Polaco, Portugués y Vasco. Normalmente no debería ser necesario hacer nada especial para conseguir que Gretl aparezca en el idioma local: el programa queda instalado con todas las traducciones y automáticamente debería aparecer en el idioma en que Vd tenga configurado Windows. Sin embargo, en algunas circunstancias puede ser necesario ajustar su configuración de idioma para obtener el efecto esperado. La configuración de idioma de Windows puede ajustarse desde el panel de control. Si no le es posible hacer que esto funcione, aquí hay un procedimiento de chequeo que puede utilizar:

  • Abra una consola de Windows (a veces llamada inexactamente "DOS box").

  • Cambie (cd) al directorio donde Gretl esté instalado. Por ejemplo, teclee

    cd c:\Archivos de programa\gretl

    y presione la tecla Enter.

  • Teclee "set LANG=" seguido por el código de dos letras de su idioma. Por ejemplo, si su idioma es el castellano, teclee

    set LANG=es

    y presione Enter. (Francés es fr, Castellano es es, Italiano es it, Polaco es pl, Portugués es pt, Alemán es de y Vasco es eu.)

  • Entonces ejecute Gretl desde la consola: tecleando

    gretlw32.exe

    y presione Enter.

Gretl debería aparecer en el idioma seleccionado, si está soportado en Windows. Si Vd. consigue que Gretl funcione de esta manera con el idioma correcto, debería ser posible conseguir el mismo efecto por medio de la correcta configuración de idioma mediante el panel de control de Windows.

Actualizaciones de Gretl

Cuendo se actualiza una versión existente de Gretl, es buena idea ejecutar antes el programa desinstalador unins000.exe, que debería Vd. encontrar en el directorio principal de Gretl. Esto evitará que gretl funcione mal debido al uso de DLLs antiguos o inscripciones anteriores en el registro de Windows. Nótese que al ejecutar el desinstalador no se tocará ningún fichero de usuario que Vd. haya creado en el directorio gretl de usuario, pero se borrará la configuración personal que haya determinado utilizando el menú 'preferencias', así que tendrá Vd. que restablecerla cuando ejecute la nueva versión del programa.

Construyendo Gretl para Windows

Si desea construir Gretl para Windows Vd. mismo, descargue el paquete de fuentes de Gretl (vaya aquí para encontrar el más reciente) y eche un vistazo en el subdirectorio win32. Aquí encontrará el archivo 'Makefile' para la versión Windows: actualmente está diseñado para cross-compilar gretl en Linux pero no debería ser muy difícil modificarlo para compilar en Windows, especialmente si Vd. utiliza el compilador mingw32.
