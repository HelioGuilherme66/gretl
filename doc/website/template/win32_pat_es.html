Gretl: Gnu Regression, Econometrics and Time-series Library

Versión para MS Windows

Requisitos del sistema

Desde la versión 1.9.4, gretl requiere Windows XP o superior y un procesador que pueda utilizar el conjunto de instrucciones SSE2. Todos los procesadores modernos soportan el uso de SSE2; no lo hacen los procesadores AMD anteriores al Athlon 64, y los procesadores Intel más antiguos que el Pentium 4. Si desea utilizar gretl en sistemas antiguos mire abajo.

Descarga

Si en Windows tiene los derechos de un super-usuario o mejores, elija un fichero de autoinstalación en la primera columna de abajo; basta con descargarlo y ejecutar el fichero exe. Esto es de aplicación para la mayoría de los usuarios.

Si Vd no tiene en Windows derechos de administrador elija un fichero zip de la segunda columna; descomprima ese fichero zip en cualquier directorio en el que Vd tenga permisos de escritura. Nota: debe preservar la estructura interna de directorios que hay dentro del archivo zip o si no gretl no funcionará. En los programas de descompresión esta opción se suele denominar "permitir carpetas". El archivo completo se encuentra en un directorio denominado gretl. Por ejemplo, si Vd descomprime el archivo zip en un directorio llamado c:\userdata, el programa GUI de gretl estará en c:\userdata\gretl\gretlw32.exe.

La versión actual "instantánea" de gretl está más actualizada que la versión oficial: normalmente suele contener correcciones de errores pero, a veces, contendrá nuevos errores. Para ver que háy de nuevo en la vesrión "instantánea", puede echar un vistazo al Historial de cambios (el elemento "in progress").

Nota: la versión de gretl de 32 bits funcionará sobre un Windows de 64 bits, pero no esto noocurrirá en el sentido contrario. Si Vd no está seguro sobre si el Windows que tiene instalado es de 64 bits o no, puede comprobarlo en la página de Microsoft knowledge base.

gretl para sistemas antiguos

Para sistemas antiguos, gretl 1.9.3 se puede ejecutar bajo Windows 2000 (y no necesita SSE2), y gretl 1.6.5 es la versión más reciente que correrá sobre Windows 98.


Soporte para varios idiomas

Ademas del Inglés, Gretl soporta el Albanés, Alemán, Checo, Chino tradicional, Español, Francés, Griego, Italiano, Polaco, Portugués, Ruso, Turco y Vasco. Normalmente no debería ser necesario hacer nada especial para conseguir que Gretl aparezca en el idioma local: el programa queda instalado con todas las traducciones y automáticamente debería aparecer en el idioma en que Vd tenga configurado Windows. Sin embargo, en algunas circunstancias puede ser necesario ajustar su configuración de idioma para obtener el efecto esperado. La configuración de idioma de Windows puede ajustarse desde el panel de control.

Actualizaciones de Gretl

Cuando se actualiza una versión existente de Gretl, es buena idea ejecutar antes el programa desinstalador unins000.exe, que debería Vd. encontrar en el directorio principal de Gretl. Esto evitará que gretl funcione mal debido al uso de DLLs antiguos o inscripciones anteriores en el registro de Windows. Nótese que al ejecutar el desinstalador no se tocará ningún fichero de usuario que Vd. haya creado en el directorio gretl de usuario, pero se borrará la configuración personal que haya determinado utilizando el menú 'preferencias', así que tendrá Vd. que restablecerla cuando ejecute la nueva versión del programa.

Construyendo Gretl para Windows

Si desea compilar Gretl para Windows Vd. mismo, descargue el paquete de fuentes de Gretl (vaya aquí para encontrar el más reciente) y eche un vistazo en el subdirectorio win32. Aquí encontrará el archivo 'Makefile' para la versión Windows: actualmente está diseñado para cross-compilar gretl en Linux pero no debería ser muy difícil modificarlo para compilar en Windows, especialmente si Vd. utiliza el compilador mingw32.
