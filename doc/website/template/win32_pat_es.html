Gretl: Gnu Regression, Econometrics and Time-series Library

Versión para MS Windows

Descarga

Gretl para Windows viene en la forma de un fichero ejecutable auto-instalable, gretl-VERSION.exe (SHORTDATE, aproximadamente 9MB). Vd. sólo tiene que descargarlo y ejecutarlo. Durante el proceso de instalación se le preguntará por el directorio donde desea instalar el programa (por defecto es c:\Archivos de programa\gretl). El programa ha sido chequeado principalmente sobre Windows XP; y debería funcionar bien sobre cualquier versión de Windows basada en NT (esto es, NT4, 2000, XP o Vista; pero no Win9X o Windows ME)

El enlace de arriba señala a la última versión oficial del programa. Otra posibilidad es descargar la versión "instantánea" de gretl: gretl_install.exe. Ésta estará normalmente más actualizada: a veces incorporará correcciones de errores, y otras veces ¡puede incorporar nuevos errores que antes no existían! Además es probable que las traducciones a idiomas diferentes del Inglés estén incompletas en esta versión. ¿Qué hay nuevo en la versión instantánea? Eche un vistazo al Historial de cambios (las cosas "en preparación").

Si Vd. desea ejecutar gretl en versiones antiguas de Windows puede utilizar la versión 1.6.5 del programa (que requiere Windows 98 o superior).

Opciones extra que es posible que Vd. desee instalar:

X-12-ARIMA (corrección de estacionalidad, modelos ARIMA)

x12a_install.exe

TRAMO/SEATS (corrección de estacionalidad, detección de outliers, interpolación de valores ausentes, modelos ARIMA)

ts_install.exe

Conjuntos de datos del libro de Wooldridge, Introductory Econometrics

wooldridge_data.exe

Conjuntos de datos del libro de Gujarati, Basic Econometrics

gujarati_data.exe

Conjuntos de datos + guiones del libro de Stock and Watson, Introduction to Econometrics

stock_watson.exe, stock_watson_2.exe,

Conjuntos de datos del libro de Davidson y MacKinnon, Econometric Theory and Methods

ETM_data.exe

Todos los de arriba son ficheros auto-instalables. Vd. debe instalar Gretl antes de instalar las opciones extra.

Los ficheros ejecutables se compilaron bajo GNU/Linux usando mingw32 y el GTK para Windows (¡gracias a Tor Lillqvist!). El programa de instalación, también software libre, es cortesía de Jordan Russell.

Soporte para varios idiomas

Ademas del Inglés, Gretl soporta el Alemán, Checo, Español, Francés, Italiano, Polaco, Portugués, Ruso, Turco y Vasco. Normalmente no debería ser necesario hacer nada especial para conseguir que Gretl aparezca en el idioma local: el programa queda instalado con todas las traducciones y automáticamente debería aparecer en el idioma en que Vd tenga configurado Windows. Sin embargo, en algunas circunstancias puede ser necesario ajustar su configuración de idioma para obtener el efecto esperado. La configuración de idioma de Windows puede ajustarse desde el panel de control.

Actualizaciones de Gretl

Cuendo se actualiza una versión existente de Gretl, es buena idea ejecutar antes el programa desinstalador unins000.exe, que debería Vd. encontrar en el directorio principal de Gretl. Esto evitará que gretl funcione mal debido al uso de DLLs antiguos o inscripciones anteriores en el registro de Windows. Nótese que al ejecutar el desinstalador no se tocará ningún fichero de usuario que Vd. haya creado en el directorio gretl de usuario, pero se borrará la configuración personal que haya determinado utilizando el menú 'preferencias', así que tendrá Vd. que restablecerla cuando ejecute la nueva versión del programa.

Construyendo Gretl para Windows

Si desea compilar Gretl para Windows Vd. mismo, descargue el paquete de fuentes de Gretl (vaya aquí para encontrar el más reciente) y eche un vistazo en el subdirectorio win32. Aquí encontrará el archivo 'Makefile' para la versión Windows: actualmente está diseñado para cross-compilar gretl en Linux pero no debería ser muy difícil modificarlo para compilar en Windows, especialmente si Vd. utiliza el compilador mingw32.
