para Mac OS X

Instalação passo a passo

A forma mais simples de instalar o Gretl para Mac OS X é baixar e instalar duas imagens (arquivos/ficheiros dmg). Em primeiro lugar instale o GTK+ Framework para o Mac OS X e em seguida baixe e instale a versão do Gretl correspondente a arquitetura do seu Mac (Intel ou PowerPC). A seguir, são apresentados os passos necessários para a instalação do Gretl em seu Mac.

Passo 0

Nos próximos passos, as escolhas apropriadas dependerão do tipo de Mac que está sendo utilizado. Se você tem dúvidas sobre isso, clique no ícone da Apple no canto superior esquerdo de sua tela/ecrã e selecione "Sobre Este Mac". Na janela que será exibida você encontrará a versão do OS X e o tipo de processador utilizado, Intel ou PowerPC (PPC).

Passo 1

Para que o Gretl funcione é necessário possuir o aplicativo X11 instalado em seu Mac. Nas versões mais recentes do OS X, o X11 vem pré-instalado, porém, caso esteja usando uma versão anterior você pode instalá-lo a partir do DVD de instalação do Mac OS X.

Como saber se o X11 está instalado? Abra o Finder, vá até Aplicativos e localize a pasta Utilitários. Se o X11 estiver instalado, ele deverá aparecer na pasta como um ícone.

Passo 2

O Gretl precisa de algumas bibliotecas adicionais para funcionar, como a biblioteca GTK. Se você estiver utilizando o Mac OS X 10.6 (Snow Leopard) ou superior em um Mac Intel, instale o GTK utilizando o arquivo GTK+.framework.dmg. Se você estiver utilizando um Mac PowerPC ou se estiver estiver utilizando uma versão mais antiga do OS X, você deve instalar essa versão do GTK (endereço alternativo).

Passo 3

Finalmente, para a instalação do próprio Gretl, baixe e instale a versão mais recente ou, alternativamente, baixe a imagem mais atual (snapshot). Ao optar por instalar a versão snapshot, você terá o Gretl mais atualizado, com algumas resoluções de bugs e novas funcionalidades. Porém, dado seu caráter experimental, o snapshot poderá introduzir novos bugs. Para saber quais novidades foram introduzidas no snapshot, consulte o Registro de alterações (no link "em progresso").

Após finalizar o download, dê um clique duplo no arquivo gretl.dmg.gz para descompactá-lo. Esse procedimento criará um arquivo dmg; dê um clique duplo para que a imagem seja montada. Em seguida, abra a imagem no Finder e arraste-a para a pasta Aplicativos. Mas atenção: se você estiver atualizando o Gretl é aconselhável que a versão anterior desinstalada (abra a pasta Aplicativos e delete o Gretl). Após finalizar a instalação você pode ejetar a imagem e deletar os arquivos gretl.dmg e gret.dmg.gz.

A imagem de disco do Gretl está documentada, de forma resumida, no arquivo/ficheiro README.pdf, que contém algumas sugestões para solucionar problemas. Caso deseje depurar o Gretl no Mac OS X, tente os seguintes passos: abra uma janela de terminal X11; altere o diretório atual (cd) para Gretl.app (no diretório onde o Gretl foi instalado); altere o diretório para Contents/Resources/bin e execute o comando

./gretl

Isto poderá fornecer alguma pista sobre possíveis erros.
