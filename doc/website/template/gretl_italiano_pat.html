Gretl è ospitato da

è un pacchetto software multi-piattaforma per l'analisi econometrica, scritto in linguaggio C. È software libero: è possibile redistribuirlo e/o modificarlo secondo i termini della Licenza Pubblica Generica GNU (GPL) pubblicata dalla Free Software Foundation.

Caratteristiche

  • Interfaccia facile e intuitiva (ora anche in francese, italiano e spagnolo, oltre che in inglese)

  • Un'ampia varietà di stimatori basati sui minimi quadrati, compresi i minimi quadrati a due stadi e i minimi quadrati non lineari

  • Singoli comandi per eseguire operazioni come test di Dickey-Fuller, test di Chow per la stabilità strutturale, autoregressioni vettoriali, stime ARMA

  • Risultati dei modelli esportabili sotto forma di tabelle o equazioni LaTeX

  • Linguaggio di scripting integrato, per eseguire comandi tramite script, oltre che tramite l'interfaccia grafica

  • Struttura loop per le simulazioni Monte Carlo e le procedure di stima iterativa

  • Controlli interattivi per personalizzare i grafici Gnuplot

  • Compatibile con GNU R per analisi statistiche ancora più approfondite

Formati di dati

Legge file nel proprio formato XML, file csv (valori separati da virgole), fogli di lavoro di Excel e Gnumeric, file BOX1, database nel proprio formato binario (che permette di archiviare dati con diverse frequenze e lunghezze delle serie) e database RATS 4. Include un database macro relativo all'economia USA. Si veda anche la pagina dei dati di gretl.

Sia che vogliate compilare dai sorgenti, sia che vogliate installare binari pre-compilati, potreste essere interessati alle dipendenze di gretl.

Gretl si basa su gnuplot per generare i grafici. L'ultima versione ufficiale di gnuplot è la 4.0 (dell'aprile 2004), disponibile su www.gnuplot.info.

Manuale

Il manuale completo di gretl è disponibile come file PDF (c. 1Mb, circa 90 pagine), impaginato in formato US letter (manual.pdf) oppure A4 (manual-a4.pdf). È disponibile anche la traduzione italiana della sezione "guida comandi" del manuale, in formato PDF (cmdbook_it.pdf), a cura di Cristian Rigamonti. È possibile scaricare il software per leggere i file PDF dal sito Adobe.

Usate gretl?

Se sì (in particolare se lo usate nel vostro lavoro o nell'insegnamento), mandatemi una email. Il riconoscimento da parte delle persone che trovano utile il programma è importante per giustificare il tempo speso a migliorarlo. Ringrazio tutti coloro che hanno già manifestato il loro interesse per gretl.
