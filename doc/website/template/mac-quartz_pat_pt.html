Instalando gretl

Obter e instalar a versão mais recente ou o "snapshot" (versão de desenvolvimento). O "snapshot" está mais atualizado: normalmente conterá correções de erros, mas por vezes poderá ter novos erros introduzidos. Para saber o que existe novo no "snapshot", dê uma vista de olhos no Registo/Registro de alterações do gretl (na entrada "in progress").

versão mais recente (SHORTDATE)

gretl-quartz.pkg

OU "snapshot" atual

gretl-quartz.pkg

Depois de obter o gretl, faça um duplo-clique no arquivodmg.gz para o descomprimir. Isto produzirá um arquivo dmg; fazer um clique-duplo nele para montar a imagem de disco. Abrir a imagem de disco no Localizador e arrastar Gretl para dentro de Aplicativos. Abrir a imagem de disco no Finder e arrastar Gretl para dentro de Applications. Mas note: se você estiver atualizando uma instalação do gretl, você deverá primeiro abrir Aplicativos com o Localizador e apagar o item do Gretl antigo.

Depois de ter instalado o gretl você pode ejetar a imagem de disco e apagar os arquivos dmg e dmg.gz.

Problemas?

A imagem de disco do gretl disk image está documentada em README-quartz.pdf, que contém algumas sugestões para despiste de problemas. Resumidamente, se você quiser despistar problemas num gretl não funcional em OS X, tente isto: abrir uma janela de terminal e correr os seguintes comandos (altere a primeira linha se o local onde instalou o gretl for outro que /Aplicativos): 

cd /Aplicativos/Gretl.app/Contents/MacOS
./gretl.sh

Isto deve mostrar alguma informação sobre o que está acontecendo de errado.

Extras opcionais

Você pode querer instalar X-12-ARIMA e/ou TRAMO/SEATS. Estes são programas especializados para análise de séries temporais com sazonalidade, em versões compatíveis com gretl.
