gretl (página principal aqui) foi desenhado para ser um pacote de econometria muito "user-friendly". Apesar de ser razoavelmente sofisticado, falta-lhe alguns dos métodos estatísticos especializados que um econometrista profissional possa desejar.

De modo a ultrapassar esta limitação, gretl oferece uma interface para o pacote de software livre de estatística, GNU R. Citando a introdução no sítio da Internet do Projecto R, "R é 'GNU S', uma linguagem e ambiente para computação e gráficos de estatística. R é semelhante ao premiado sistema S, que foi desenvolvido em Bell Laboratories de John Chambers e outros".

A partir de gretl, é possível guardar o conjunto de dados corrente num formato adequado para análise usando R; também é possível ("Iniciar GNU R" no menu "Utilitários" do gretl) lançar uma sessão de R em que o conjunto de dados corrente fica imediatamente carregado no espaço de trabalho do R.

Para saber mais sobre o GNU R, recomendo começar com  An Introduction to R, disponível aqui como um ficheiro PDF. Para mais informação ou para fazer o download do R, visite cran.r-project.org.
