Terminología

En el lenguaje de gretl, una base de datos es una colección de series (normalmente grande) en la que puede haber datos mezclados de diferentes frecuencias y rangos de observación. Los datos se almacenan en formato binario, como números "flotantes" de precisión simple [Nota]. Se puede acceder a dichos ficheros por medio del menú Archivo/Revisar base de datos de gretl. Desde allí Vd. puede importar al espacio de trabajo de gretl las series que desee. Los ficheros de bases de datos gretl tienen extensión .bin, y van acompañados de un fichero de índice (con extensión .idx) que contiene la información sobre las series.

Por otro lado, un fichero de datos gretl, es una colección de series que es homogéneo en cuanto a frecuencia y rango de observación (aunque puede haber valores ausentes para algunas variables en algunas observaciones). Este tipo de ficheros puede leerse directamente y ser cargado dentro del espacio de trabajo de gretl. Los ficheros de datos de gretl suelen tener extensión .gdt ; por defecto están en formato XML.

Para más detalles, consulte por favor el manual de gretl.

1. Bases de datos de gretl

La distribución de gretl contiene una base de datos de ejemplo, el conjunto de "Indicadores del ciclo de negocios" del U.S. Department of Commerce. Hay algunas bases de datos más disponibles para su descarga de forma gratuita, o para acceso en línea. Nótese que con gretl 0.40 y superiores Vd. puede acceder a estas bases de datos adicionales por medio del menú Archivo/Revisar base de datos/sobre servidor (siempre que Vd. esté conectado a internet).

Tabla Mundial de Penn

PWT56 es un conjunto de datos de panel bastante rico, recoge 152 países a lo largo de los años 1950 a 1992. El paquete para gretl comprende tres ficheros de datos:

  • pwt56.gdt: pwt56 entera. Este conjunto de datos tiene muchas observaciones ausentes.

  • pwt56_60_89.gdt: es un conjunto de datos de panel de 120 países durante los 30 años 1960-89, y contiene 20 variables. Este conjunto de datos no tiene observaciones ausentes.

  • pwt56_1985.gdt: Hay más datos disponibles para 1985 que para ningún otro año, así que también construí un fichero de datos puro de sección cruzada para los 152 países para este año. Este fichero tiene algunas observaciones ausentes, pero éstas no son tan difíciles de tratar en una sección cruzada como en un panel.

El paquete incluye la documentación completa y guiones ('scripts') de ejemplo para analizar los datos.

Le sugiero que consulte el manual de gretl como guía para trabajar con datos de panel.

3. Datos del libro de Wooldridge

Desde la versión 1.0, gretl ofrece soporte para los conjuntos de datos que se usan en el libro de Jeffrey Wooldridge, Introductory Econometrics: A Modern Approach. En swcollege.com puede encontrarse un paquete que contiene los datos en el formato XML nativo de gretl.

4. Utilidades para el manejo de datos

Además de ofrecer bases de datos especialmente preparadas, gretl contiene algunos guiones ('scripts') que pueden usarse para crear bases de datos usando datos disponibles vía internet. Estos se encuentran en el subdirectorio utils del paquete de fuentes de gretl (ver la página principal) de gretl. Para ejecutar esos guiones, Vd. necesita tener instalado perl y estar conectado a internet.

Nota: Las bases de datos binarias mencionadas arriba fueron preparadas en un PC ix86. Si Vd. desea utilizar los datos sobre otra arquitectura , por favor, hágamelo saber; puedo proporcionarle las bases de datos en formato ASCII con un pequeño programa en C que extraiga los números en formato binario.
