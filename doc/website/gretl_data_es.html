Terminología

En la jerga de gretl, una base de datos es una colección de series (normalmente grande) en la que puede haber datos mezclados de diferentes frecuencias y rangos de observación. Los datos se almacenan en formato binario, como números "flotantes" de precisión simple [Nota]. Se puede acceder a dichos ficheros por medio del menú Archivo/Revisar base de datos de gretl. Desde allí Vd. puede importar al espacio de trabajo de gretl las series que desee. Los ficheros de bases de datos gretl tienen extensión .bin, y van acompañados de un fichero de índice (con extensión .idx) que contiene la información sobre las series.

Por otro lado, un fichero de datos gretl, es una colección de series que es homogéneo en cuanto a frecuencia y rango de observación (aunque puede haber valores ausentes para algunas variables en algunas observaciones). Este tipo de ficheros puede ser leído directamente y cargarse dentro del espacio de trabajo de gretl. Los ficheros de datos de gretl suelen tener extensión .gdt ; por defecto están en formato XML, de acuerdo con la definición de tipo de documento (DTD) que se encuentra en gretldata.dtd.

1. Bases de datos de gretl

La distribución de gretl contiene una base de datos de ejemplo, una colección de series temporales macroeconómicas de Estados Unidos obtenida del Banco de la Reserva Federal de San Luis. Hay también otras bases de datos disponibles gratuitamente para su descarga o acceso en línea. Note que con gretl 0.40 o superior Vd puede acceder a estas bases de datos adicionales por medio del menu Archivo/Bases de datos/sobre servidor (siempre que Vd esté conectado a internet).

2. Ficheros de datos de libros de texto

Hay ficheros de datos (y en algunos casos, guines de isntrucciones) disponibles para varios manuales de econometría, como se muestra en la tabla de abajo. Instalación de los datos de los libros de texto

Con los ficheros de auto-instalación de Windows, basta con que los descargue al escritorio y pulse dos veces sobre el icono de instalación. (Una vez que se hayan instalado puede Vd. borrar el fichero instalador.)

Los paquetes de datos comprimidos deberían desempaquetarse en el directorio "data" de gretl. Por ejemplo si gretl se instala en /usr Vd. podría isntalar los datos de Stock y Watson así (como usuario root):

cd /usr/share/gretl/data
wget http://ricardo.ecn.wfu.edu/pub/gretl/stock_watson.tar.gz
tar xvfz stock_watson.tar.gz
rm stock_watson.tar.gz

El paquete de guiones de Stock y Watson debería desempaquetarse en /usr/share/gretl/scripts (o el lugar correspondiente si gretl no se instala en /usr).

Tabla Mundial de Penn

PWT56 es un extenso conjunto de datos de panel macroeconómicos, que engloba 152 países durante los años desde 1950 hasta 1992. El paquete de gretl comprende tres archivos:

  • pwt56.gdt: la pwt56 completa. Este conjunto de datos tiene muchas observaciones ausentes.

  • pwt56_60_89.gdt: un conjunto de datos de panel de 120 países para los 30 años desde 1960 a 1989, que contiene 20 variables. Este conjunto de datos no tiene observaciones ausentes.

  • pwt56_1985.gdt: Hay más datos para 1985 que para cualquier otro año, así que también preparé un fichero de datos de sección cruzada puro para todos los 152 países para este año. Este archivo tiene algunas observaciones ausentes, pero no será tan dificl tratarlas en una sección cruzada como en un panel.

El paquete contiene información completa y guiones de ejemplo para analizar los datos.

Si Vd. usa gretl en su docencia, es posible que desee organizar una colección personalizada de ficheros de datos para que usen sus estudiantes. Eche un vistazo aquí para más información.

5. Utilidades para el manejo de datos

Además de ofrecer bases de datos especialmente preparadas, gretl viene con algunos programas que pueden usarse para crear bases de datos usando datos disponibles vía internet. Estos se encuentran en el subdirectorio utils del paquete fuente de gretl (ver la página principal) de gretl.
