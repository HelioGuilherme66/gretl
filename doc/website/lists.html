gretl mailing lists

At present there are three gretl-related mailing lists.

gretl announce

This mailing list sends out a short announcement each time a new version of gretl is released. Please note that this list is for one-way communication only; members cannot post to it.

gretl users

This list is intended for discussion between gretl users. It is an appropriate forum for raising bug reports, requesting features, making suggestions, asking how things work, and so on. To post to the list, you must be subscribed. The proceedings are archived.

gretl devel

This list is intended for discussion about bug fixing, planning of new features, and coordination of work between gretl developers. To post to the list, you must be subscribed. The proceedings are archived.

links
