Terminologia

Na linguagem corrente do gretl, uma database (base de dados) é uma colecção de séries temporais (normalmente grande) que podem ser um misto de frequências e intervalos de observações. Os dados estão guardados em formato binário, como sendo de vírgula flutuante (float) de precisão singular [Nota]. Acede-se as esses ficheiros por meio do menu Ficheiro/Procura bases_de_dados, que permite seleccionar as séries e importar para o espaço de trabalho do gretl.  As bases de dados do gretl têm uma extensão .bin e são acompanhadas por um ficheiro de índice (extensão .idx) que contém informação sobre a série.

Por outro lado, um ficheiro de dados (datafile), é uma colecção de séries que é homogénea no que diz respeito à frequência e intervalo de dados (embora possam haver valores omissos para algumas variáveis em algumas observações). Este tipo de ficheiro pode ser lido directamente para o espaço de trabalho do gretl. Tipicamente estes ficheiros têm uma extensão .gdt; estão em formato XML, e de acordo com o standard "Document Type Definition" (DTD) dado em gretldata.dtd.

Para mais detalhes, consultar o manual do gretl.

1. Bases de Dados para o gretl

A distribuição de gretl contém uma base de dados de exemplos; uma colecção de séries temporais de informação macro-económica da Reserva Federal de St. Louis nos E.U.A.. Outras bases de dados estão disponíveis para download ou acesso on-line. A partir da versão 0.40 do gretl, é possível aceder a estas bases de dados adicionais a partir do menu Ficheiro/Procura bases_de_dados/em servidor de bases_de_dados (desde que se esteja ligado à Internet).

2. Conjuntos de Dados de livros de texto

Com o pacote base do gretl vêm fornecidos os conjuntos de dados e scripts sobre o livro Introductory Econometrics with Applications de Ramu Ramanathan, e também, uma selecção de dados do livro Econometric Analysis de William Greene. Indicam-se outros conjuntos de dados adicionais que estão disponíveis:

Wooldridge: gretl fornece suporte aos conjuntos de dados utilizados no livro de texto de Jeffrey Wooldridge Introductory Econometrics: A Modern Approach. O conjunto de dados está disponível como ficheiro auto-executável em Windows wooldridge_data.exe, ou como arquivo comprimido wooldridge.tar.gz.

Gujarati: gretl suporta os conjuntos de dados do livro Basic Econometrics de Damodar Gujarati. Ver nesta página mais detalhes.

Stock and Watson: gretl suporta os conjuntos de dados do livro Introduction to Econometrics de James Stock e Mark Watson. Também são fornecidos os scripts de replicação dos exemplos. O pacote está disponível como um ficheiro auto-instalável em Windows, stock_watson.exe, ou como arquivo comprimido, stock_watson.tar.gz.

Davidson and MacKinnon: gretl suporta os conjuntos de dados do livro deles, Econometric Theory and Methods. Os conjuntos de dados encontram-se num ficheiro auto-instalável em Windows, ETM_data.exe, ou em arquivo comprimido ETM.tar.gz.

No que diz respeito aos ficheiros auto-instaláveis em Windows, fazer download deles para o Ambiente de Trabalho e fazer duplo-click no ícone. (Uma vez instalados, pode-se apagar os instaladores).

Os arquivos comprimidos com gzip devem ser descomprimidos na pasta "share" do gretl. Por exemplo, se gretl está instalado em /usr, para instalar os dados e script de Stock e Watson basta proceder como abaixo se indica (com acesso root):

cd /usr/share/gretl
wget http://ricardo.ecn.wfu.edu/pub/gretl/stock_watson.tar.gz
tar xvfz stock_watson.tar.gz
rm stock_watson.tar.gz

Penn World Table

PWT56 é um rico painel de dados do tipo macro-económico, que abranje 152 países ao longo dos anos 1950 a 1992. O pacote do gretl contém três ficheiros de dados principais:

  • pwt56.gdt: O pwt56 completo. Este conjunto de dados contém muitos valores omissos.

  • pwt56_60_89.gdt: Um painel de dados com 120 países sobre 30 anos que vão de 1960 a 1989, contém 20 variáveis. Este conjunto de dados não tem valores omissos.

  • pwt56_1985.gdt: Dado que em 1985 estão disponíveis mais dados que para outros anos, construí este conjunto de dados que contém dados sobre várias secções de todos os 152 países. Este conjunto de dados tem alguns valores omissos, mas será mais fácil lidar com eles do que no conjunto completo.

Este pacote inclui a documentação completa e exemplos de scripts para analisar os dados.

Para obter mais informação em como lidar com dados de painel, recomendo consultar o manual do gretl.

Se usa o gretl no ensino, poderá ser útil preparar colecções de dados (ou scripts) para os estudantes acederem. Ver nesta página instruções detalhadas em como o fazer.

5. Utilitários para tratamento de Dados

Para além de oferecer bases de dados pré-preparadas, o gretl vem com alguns scripts que podem ser utilizados para criar bases de dados a partir de dados disponíveis na Internet. Estes podem ser encontrados na subdirectoria utils do pacote de código-fonte do gretl (ver página principal do gretl). Para executar os scripts tem que ter instalado o perl no computador e estar conectado à Internet.
