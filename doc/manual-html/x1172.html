NLS dialog window

It is probably most convenient to compose the commands for NLS estimation in the form of a gretl script but you can also do so interactively, by selecting the item "Nonlinear Least Squares" under the Model menu. This opens a dialog box where you can type the function specification (possibly prefaced by genr lines to set the initial parameter values) and the derivatives, if available. An example of this is shown in Figure 7-1. Note that in this context you do not have to supply the nls and end nls tags.
